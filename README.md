# Landing Page - Alessandra Figueiredo | Fisioterapia e Massoterapia

![Status do Projeto](https://img.shields.io/badge/Status-Concluído-brightgreen)

## 📄 Sobre o Projeto

Este projeto é uma **Landing Page** responsiva desenvolvida para apresentar o portfólio de serviços da Dra. Alessandra Figueiredo, uma profissional com vasta experiência em fisioterapia e massoterapia atuante na região de Juatuba e Mateus Leme/MG.

O objetivo do site é facilitar o acesso às informações sobre os tratamentos oferecidos, destacar os diferenciais clínicos da profissional e fornecer um canal direto e rápido para agendamento de consultas via WhatsApp.

## ✨ Funcionalidades

- **Apresentação Profissional:** Seção detalhando a formação e a experiência da fisioterapeuta.
- **Catálogo de Tratamentos:** Exibição clara dos serviços prestados (Osteopatia, Ventosa, Massoterapia, Pilates).
- **Diferenciais Clínicos:** Destaque para técnicas especializadas (Pilates Terapêutico, Técnica Tui-ná MTC e Biomagnetismo).
- **Chamadas para Ação (CTAs):** Botões estratégicos direcionando o usuário diretamente para o WhatsApp da clínica.
- **Navegação Suave (Smooth Scroll):** Implementação em JavaScript para uma navegação mais agradável entre as seções da página.
- **Rodapé Informativo:** Contatos completos (Telefone, E-mail, Instagram) e integração com link para o Google Maps com o endereço do consultório.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias web padrão:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização e responsividade (através do arquivo `style.css`).
- **JavaScript (Vanilla):** Interatividade da página, como a rolagem suave dos links (Smooth Scroll).
